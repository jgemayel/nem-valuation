# Newmont (NEM) DCF Valuation & Scenario Analysis

Interactive valuation dashboard for Newmont Corporation (NYSE: NEM), the world's largest gold miner.

## Live Dashboard
**https://jgemayel.github.io/nem-valuation/**

## What's Included

### Interactive Dashboards
- **5-Year Total Return Analysis** - Shows your actual dollar returns under 6 gold price scenarios ($4k to $10k), including dividends, buyback accretion, and terminal re-rating
- **DCF Scenario Analysis** - Full discounted cash flow model with WACC sensitivity, terminal value via exit multiple and perpetuity growth methods

### Valuation Memo (Word Document)
Full investment banking-style DCF memo with:
- Plain-English investment summary
- 5-year revenue projections with gold price assumptions
- Operating margin estimates and EBITDA bridge
- Free cash flow waterfall
- WACC derivation (CAPM)
- Terminal value (dual methodology)
- Sensitivity table (WACC x terminal growth)
- Market comparison and verdict
- Key risks and model-breaking assumptions

Download: [Newmont_NEM_DCF_Valuation_Memo.docx](./public/Newmont_NEM_DCF_Valuation_Memo.docx)
